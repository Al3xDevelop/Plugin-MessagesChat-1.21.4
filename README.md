#Plugin-MessagesChat-1.21.4
Инструкция по компиляции плагина для Minecraft через Maven

Установленная Java Development Kit (JDK) версии 21

Установленный Apache Maven

Установите исходный код вашего плагина

Соберите плагин через команду в терминале "mvn package"

Готовый плагин будет лежать в папке c:/users/ваше имя/IdeaProjects/StarsMessagesChat/Target

Плагин поддерживается на версии 1.21.4 paper minecraft (java) Для работы плагина понадобится: ядро сервера paper версии 1.21.4
